Opdracht

Ontwikkelen van een REST API voor het herkennen van duplicate records binnen een dataset op basis van Machine Learning. Verschillende endpoints voor trainen en herkennen en een UI waar mensen met hun Salesforce account (oAuth) kunnen inloggen om op hun records in Salesforce het model te trainen en gebruiken via de REST API.

Er wordt gebruik gemaakt van bestaande techniek voor de ML learning. Hier hoeft geen onderzoek naar te worden gedaan. 

Ruud is de eerste week op ons kantoor in Lissabon maar is uiteraard via online meetings beschikbaar om de kick-off te begeleiden.
Technische Constraints/ Wensen

   - REST API in NestJS (Node framework),
   - matching via RecordLinkage (python),
   - UI in Vue3 (composition API).
